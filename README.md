# CodeAlpha C++ Internship Tasks

This repository contains the tasks completed as part of the **CodeAlpha C++ Programming Internship**.

---

## Task 1: CGPA Calculator
This project is a C++ program that calculates the CGPA of a student based on the marks or grades entered for multiple subjects.  
It takes user input, processes the data, and displays the final CGPA.

**Concepts Used:**
- Loops
- Conditional statements
- User input handling

---

## Task 2: Login & Registration System
This project implements a basic Login and Registration System using C++.  
Users can register by creating credentials and then log in using valid username and password.

**Concepts Used:**
- File handling
- Conditional logic
- Basic authentication system

---

## Task 3: Banking System
This project simulates a simple Banking System using C++.  
It allows users to perform basic banking operations such as account creation, deposit, withdrawal, and balance inquiry through a menu-driven program.

**Concepts Used:**
- Functions
- Menu-driven programming
- Real-world application logic
